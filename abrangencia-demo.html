Implementação da API Google Maps
Passos necessários:
  1. Criar conta no Google Cloud Platform
  2. Ativar a Maps JavaScript API
  3. Gerar uma API Key
  4. Configurar restrições de domínio
  5. Substituir "YOUR_API_KEY" pela chave real
Funcionalidades implementadas:
  • Mapa centralizado em Minas Gerais
  • Marcadores das 8 bases
  • Clusters automáticos
  • InfoWindows com informações detalhadas
  • Estilo personalizado
